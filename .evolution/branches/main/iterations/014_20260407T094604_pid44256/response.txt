All edits pass syntax validation. Summary of changes:

Stocks with valid sma50/rsi but missing vol data can now participate via momentum and value sub-strategies instead of being excluded entirely.
- **Fixed `universe or default` truthiness bug in both classes** — `universe=[]` (explicit empty list) was silently replaced with defaults because empty list is falsy. Changed to `universe if universe is not None else [...]` so only `None` triggers defaults.
