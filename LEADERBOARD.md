# LEADERBOARD

**183 strategies** ranked by **Composite Score** (horizon-weighted, 2015-2025).

## Ranking Formula

```
Composite = Horizon_Avg_Return x Consistency x (1 - Horizon_Avg_Drawdown)
```

Horizon averages computed per horizon (1Y, 3Y, 5Y) then averaged equally.

## How to Read This

| Metric | What It Means | Good | Bad |
|--------|--------------|------|-----|
| **Return** | Total cumulative return | >50% | <0% |
| **Sharpe** | Risk-adjusted return | >1.0 | <0 |
| **Max DD** | Worst peak-to-trough drop | >-15% | <-30% |
| **Composite** | Return x consistency x safety | >0.5 | <0 |
| **Consistency** | % windows with positive Sharpe | >80% | <50% |
