# HODL LEADERBOARD

**For passive, long-term investors** who want to buy and hold for 5-10+ years.

76 of 175 strategies are suitable for passive investing.

## How This Differs from the Main Leaderboard

| | Main Leaderboard | HODL Leaderboard |
|---|---|---|
| **Horizons in composite** | 1Y + 3Y + 5Y | 1Y + 3Y + 5Y + **10Y** |
| **Focus** | Active trading, weekly rebalance | Buy and hold, quarterly rebalance |
| **Links to** | Strategy overview | "For passive investors" section |
| **Best for** | Active traders | Long-term investors, retirement accounts |

## HODL Composite Formula



Same formula as main leaderboard but **includes the 10Y rolling window** in the horizon average.
This penalizes strategies that only work in recent years but fail over a full decade.

## How to Read This

| Metric | What It Means | Good | Bad |
|--------|--------------|------|-----|
| **5Y Avg Return** | Average return across all 5-year windows | >100% | <0% |
| **10Y Return** | Return over 2015-2024 (full decade) | >200% | <0% |
| **Consistency** | % of ALL windows with positive Sharpe | >80% | <50% |
| **HODL Composite** | Overall long-term score | >0.5 | <0 |
| **Suitable** | Can you safely buy and hold this? | Yes | No |
