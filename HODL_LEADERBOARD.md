## How This Differs from the Main Leaderboard

| | Main Leaderboard | HODL Leaderboard |
|---|---|---|
| **Horizons in composite** | 1Y + 3Y + 5Y | 1Y + 3Y + 5Y + **10Y** |
| **Focus** | Active trading | Buy and hold, quarterly rebalance |
| **Links to** | Strategy overview | "For passive investors" section |

## HODL Composite Formula

```
HODL Composite = Horizon_Avg_Return x Consistency x (1 - Horizon_Avg_Drawdown)
```

Same as main but **includes the 10Y window**.

## How to Read This

| Metric | What It Means | Good | Bad |
|--------|--------------|------|-----|
| **5Y Avg Return** | Average across 5-year windows | >100% | <0% |
| **10Y Return** | Full decade 2015-2024 | >200% | <0% |
| **Consistency** | % ALL windows positive Sharpe | >80% | <50% |
| **HODL Composite** | Long-term score | >0.5 | <0 |
| **Suitable** | Safe to buy and hold? | Yes | No |
